## Start here

- **Use a product** → [dirgha.ai](https://dirgha.ai)
- **Read the source** → pick a repo above
- **Contribute code** → each repo has a `CONTRIBUTING.md` and `docs/` folder with recipes
- **Sponsor the work** → [dirgha.ai/contribute](https://dirgha.ai/contribute) (GitHub Sponsors, OpenCollective, or Bitcoin)
- **Report a security issue** → `security@dirgha.ai` (see `SECURITY.md` in any repo)

## How we work

- **Bootstrapped.** The pace is the pace that shipping sustainable code allows.
- **Open where it helps, proprietary where it's needed.** The CLI, SDKs, and protocol layers are open source. The hosted platform, models, and brand are proprietary to Dirgha LLC.
- **Sovereign by default.** Everything we build should run on your own machine with your own keys if you want it to.
- **We ship code, not promises.** Releases, changelogs, and tests are the track record.
